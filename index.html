G+ Commenter

What is it?

G+ Commenter is a small component that you can add to your Website/Blog. This greatly simplifies the commenting in your blog.

Is it for the entire blog or the homepage?

It serves a new set of comments for each unique link of your blog/website.

What is the advantage?

Nothing big. You can write a commenting module or you could include simple G+ Commenter js & css file in your homepage and let it take care of the commenting for you. Simple and straight forward.

Want to explore the component(...ish) 

    > git clone ajainarayanan/GplusCommenter
    > cd GplusCommenter
    > npm install
    > grunt
    > nodemon server.js/node server.js
    > Go to http://localhost:3000/ and you should see
    this page with the sidebar attached to the page.
